# TERM_TYPE

Terminal typing speed test built with C++,FTXUI and CMake.

## Build
```bash
git clone https://github.com/meysaw/term-type.git
```
```bash
cd term_type
```
```bash
mkdir build && cd build
```
```bash
cmake ..
```
```bash
make
```
```bash
./term_type
```
## Controls

- R to restart
- Esc to quit
